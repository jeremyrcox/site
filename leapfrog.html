ABOUT THE PROJECT

This was two projects in one. First, I created a web-based application that Leapfrog uses to manage projects. Employees are able to track information and share select information with clients. This application was built using Laravel 4 PHP framework with MySQL and HTML, CSS, JavaScript with jQuery and Twitter Bootstrap.

We also created a new custom WordPress website with three color schemes corresponding to the three business lines. The site is responsive and uses SVG graphics and CSS3 animations.
